Value Class (AutoGrad Engine)

The Value class is a computational unit that stores data and gradients, and automatically computes derivatives using backpropagation.

class Value { constructor(data, children=[], op='') { this.data = data; this.grad = 0; this._backward = () => {}; this._prev = children; this._op = op; } // Forward: y = this + other add(other) { out = new Value(this.data + other.data); out._backward = () => { this.grad += out.grad; other.grad += out.grad; }; return out; } // Forward: y = this * other mul(other) { out = new Value(this.data * other.data); out._backward = () => { this.grad += other.data * out.grad; other.grad += this.data * out.grad; }; return out; } // Backpropagation with Topological Sort backward() { topo = []; visited = new Set(); // Build computational graph // Compute gradients in reverse order } }

Activation Functions

ReLU

f(x) = max(0, x)

Gradient: 1 if x > 0, else 0

Tanh

f(x) = (e^x - e^-x) / (e^x + e^-x)

Gradient: 1 - tanh²(x)

Sigmoid

f(x) = 1 / (1 + e^-x)

Gradient: o(1-o)

Optimizer Formulas

SGD - Stochastic Gradient Descent

θ = θ - lr × ∇θ

Update parameters based on raw gradients. lr = learning rate, ∇θ = gradient.

SGD + Momentum

v = βv + (1-β)∇θ
θ = θ - lr × v

Adds momentum to accelerate convergence. β = momentum factor (usually 0.9).

Nesterov Accelerated Gradient

v = βv + ∇θ(f(θ - βv))
θ = θ - lr × v

Look-ahead gradient for more accurate updates.

RMSProp

E[g²] = βE[g²] + (1-β)g²
θ = θ - lr × g / √(E[g²] + ε)

Adaptive learning rate based on moving average of squared gradients.

Adam - Adaptive Moment Estimation

m = β₁m + (1-β₁)g (first moment)
v = β₂v + (1-β₂)g² (second moment)
m̂ = m / (1-β₁^t), v̂ = v / (1-β₂^t)
θ = θ - lr × m̂ / (√v̂ + ε)

Combines momentum and RMSProp. Default: β₁=0.9, β₂=0.999, ε=1e-8

AdamW - Adam with Weight Decay

g = ∇θ + λθ (L2 regularization)
(then apply Adam update with g)

Adam with proper weight decay (applied to gradient, not parameters).

Loss Functions

Mean Squared Error (MSE)

L = (1/n) Σ(y - ŷ)²

Regression tasks, penalizes large errors heavily.

Cross-Entropy

L = -Σ y·log(ŷ)

Classification tasks, probabilistic outputs.

Binary Cross-Entropy

L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]

Binary classification, sigmoid activation.

Parameter Reference

Parameter Default Description
learning_rate (lr) 0.01 - 0.1 Step size for parameter update
beta1 (β₁) 0.9 Decay rate for first moment (momentum)
beta2 (β₂) 0.999 Decay rate for second moment (RMS)
momentum (β) 0.9 Velocity in SGD momentum
epsilon (ε) 1e-8 Numerical stability constant
weight_decay (λ) 0.01 L2 regularization strength
